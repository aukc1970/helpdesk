Поле

Евгений Ворожбянов
  • 07/09/2017 14:12
  • Обновлено

 

  1. Настройка поля формы input
  2. Настройка поля формы select
  3. Настройка поля формы checkbox, radio

1. Для настройки элемента формы input вам необходимо выбрать этот элемент, потом зайти в вкладку "Настройки" в блок "Поле"

Свернуть/развернуть блок: Нажмите на название этого блока

Имя: Указывается имя поля

Type: Устанавливается будет ли поле видимым(text) или скрытым(hidden)

Подсказка: Здесь вы можете установить текст подсказки поля

Маска: Выбирается одна из доступных масок поля

Обязательно поле: Устанавливается будет ли поле обязательным для заполнения или нет

2. Для настройки элемента формы select вам необходимо выбрать этот элемент, потом зайти в вкладку "Настройки" в блок "Поле"

Свернуть/развернуть блок: Нажмите на название этого блока

Имя: Указывается имя поля select

Список option: Список опций selecta 

Добавить: Добавление новой опции

После нажатия на кнопку "Добавить" вам нужно ввести значение опции и ее имя

Удаление: Удаление опции secelect

Переименовать: Для того, чтобы переименовать опцию, кликните по ее имени

3. Для настройки элемента формы checkbox или radio вам необходимо выбрать этот элемент, потом зайти в вкладку "Настройки" в блок "Поле"

Свернуть/развернуть блок: Нажмите на название этого блока

Имя: Указывается имя поля checkbox или radio

Value: Значение параметра value checkbox или radio

Отмечен: Устанавливается, будет ли checkbox или radio отмечен по умолчанию или нет

Обязательное: Устанавливается, будет ли это поле обязательным или нет
