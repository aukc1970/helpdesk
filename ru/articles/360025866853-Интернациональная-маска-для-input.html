Интернациональная маска для input

Евгений Ворожбянов
  • 04/08/2019 10:54
  • Обновлено

Пример формы с интернациональной маской для input.

Код страны будет определяться автоматически на основе IP адреса, но так же можно вручную выбрать нужную страну.

Для ее создания вам необходимо в форму добавить элемент "Блок"

После этого внутрь этого блока добавьте элемент "input"

Создайте у этого элемента "input" дополнительный класс, с любым именем. 

Например назовите его "intl", после этого уберите у него внешний отступ сверху.

Выберите элемент "Блок" и уже у него задавайте нужный внешний отступ.

Выберите input и в вкладке "Настройки" в блоке "Поле" укажите:

поле "Имя" - "phone"

поле "Подсказка" - "Телефон"

поле "Маска" - "Нет"

В блоке "Атрибуты" установите поле "id" - "phone". Так же добавьте атрибут "type" с значением "tel"

Теперь вам необходимо добавить два текстовых поля. Одно будет для вывода правильно заполненного поля(его потом скроем, но для виджета, необходимо чтобы он был добавлен в форму), другое для вывода ошибки.

Условно назовем один элемент "Правильно" другой "Ошибка". Не имеет значение, какой текст вы внесете в эти текстовые элементы.

Выберите текстовый элемент "Правильно" и в вкладке "Настройка" в блоке "Атрибуты" установите ему "id" - "valid-msg" и добавьте дополнительный класс "hide"

То же самое сделайте с текстовым элементом "Ошибка", только у него нужно будет изменить "id" на "error-msg". Так же добавьте ему дополнительный класс "hide"

Выберите текстовый элемент "Ошибка" и назначьте ему нужные стили в вкладке "Стиль".

Теперь, вам необходимо спрятать текстовый элемент "Правильно". Для этого выберите его и в вкладке "Настройка" в поле "Показать" укажите спрятать на всех экранах. 

Он должен исчезнуть с холста, но должен быть в вкладке "Структура"

Теперь вам необходимо загрузить все файлы относящиеся к этому виджету. Файлы будут прикреплены к этой статье внизу.

В файловом менеджере у вас должны быть три папки "css", "img", "js". Если, какой-то папки нет, то создайте ее.

Структура папок:

img:

- flags.png

css:

- intlTelInput.min.css

js:

- intlTelInput.min.js

- utils.js

Теперь необходимо добавить вставку кода в "Head"

Код для вставки:

<link rel="stylesheet" href="css/intltelinput.min.css">
<style>
.hide{display: none;}
</style>

Код для вставки в "Конец Body":

<script src="js/intltelinput.min.js"></script>

<script>
var input = document.querySelector("#phone");
errorMsg = document.querySelector("#error-msg"),
validMsg = document.querySelector("#valid-msg");

// Error messages based on the code returned from getValidationError
var errorMap = [ "Неправильный номер", "Неправильный код страны", "Короткий номер", "Длинный номер", "Неправильный номер"];

// ----
var intl = window.intlTelInput(input, {
utilsScript: "js/utils.js",
onlyCountries: ['RU', 'UA', 'KZ', 'KG', 'UZ', 'TJ', 'TM', 'AZ', 'GE', '', 'AM', 'LT', 'LV', 'EE', 'FI', 'NO'],
initialCountry: "auto",
geoIpLookup: function(success, failure) {
$.get("https://ipinfo.io", function() {}, "jsonp").always(function(resp) {
var countryCode = (resp && resp.country) ? resp.country : "";
success(countryCode);
});
},

},
);

var reset = function() {
input.classList.remove("error");
errorMsg.innerHTML = "";
errorMsg.classList.add("hide");
validMsg.classList.add("hide");
};

// Validate on blur event
input.addEventListener('blur', function() {
reset();
if(input.value.trim()){
if(intl.isValidNumber()){
validMsg.classList.remove("hide");
}else{
input.classList.add("error");
var errorCode = intl.getValidationError();
errorMsg.innerHTML = errorMap[errorCode];
errorMsg.classList.remove("hide");
}
}
});

// Reset on keyup/change event
input.addEventListener('change', reset);
input.addEventListener('keyup', reset);
</script>

Все готово, сохранитесь и перейдите в полный просмотр.

Список стран, можно изменить. Для этого вам необходимо убрать или добавить 2х буквенный код страны в коде, который добавляется в "Конец Body" в строчке "onlyCountries: ['RU', 'UA', 'KZ', 'KG', 'UZ', 'TJ', 'TM', 'AZ', 'GE', '', 'AM', 'LT', 'LV', 'EE', 'FI', 'NO'],"

Например, для того, чтобы было только три страны, достаточно оставить:

"onlyCountries: ['RU', 'UA', 'KZ'],"

Если необходимо добавить новую страну, тогда вам нужно будет вставить 2х буквенный код этой страны, для этого перейдите на сайт https://geo.koltyrin.ru/kody_stran_mira.php, там вы найдете нужную страну и ее 2х буквенный код.

Например, добавим к примеру выше еще страну 

Тогда строчка будет выглядеть:

"onlyCountries: ['RU', 'UA', 'KZ', 'AT'],"
