Способы передачи параметров товара

Евгений Ворожбянов
  • 14/08/2019 11:45
  • Обновлено

Когда необходимо зафиксировать через форму определенные свойства товара или его параметры. Чтобы потом эти данные отображались  в карточке товара в CMS Hollpee. Для того, чтобы все это реализовать, есть несколько способов.

1. Использовать динамические поля. Подробнее прочитать можете по этой ссылке. 

2. Использовать модальное окно совместно с заполненным полем "Имя формы".

Для этого вам необходимо создать модальное окно, в котором будет ваша форма. После этого выбрать элемент "Кнопка", потом в вкладке "Настройки" в блоке "Клик по элементу" установить это модальное окно, так же в поле "Имя формы" указать значение, которое вы хотите передать вместе с формой. В примере ниже есть несколько одинаковых велосипедов, но разных по цвету. По этому в этом поле вводится цвет велосипеда.

Второй товар

И так далее для каждого товара.

Теперь когда покупатель кликнет на любую из кнопок, у него откроется то самое модальное окно с формой. Он его заполняет, а в карточке товара будет отображаться цвет выбранного товара.
