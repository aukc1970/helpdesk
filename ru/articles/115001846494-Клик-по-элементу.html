Клик по элементу

Евгений Ворожбянов
  • 07/09/2017 14:05
  • Обновлено

В редакторе вы можете назначать разные действия при клике на элемент. Для этого вам нужно выбрать элемент перейти в вкладку "Настройки" и в блоке "Клик по элементу" выбрать нужное действие при клике

 Доступные варианты:

Нет: Значение по умолчанию. Все выключено, при клике ни чего не произойдет

Страница: Указываете, чтобы при клике на элемент открывалась страница 

Секция: Плавный переход к указанной секции, обычно используется в меню навигации

Модальное окно: Указываете, чтобы при клике на элемент открывалось модальное окно. В поле "Имя формы" можно указать имя формы, оно будет отображаться в информации о лиде в CMS Hollpee

Ссылка: Указываете, чтобы при клике на элемент открывалась ссылка

Контакт: Указываете, чтобы при клике на элемент происходил набор телефона, WhatsApp или Skype

Магазин: Указываете, чтобы при клике на элемент в CMS Hollpee передавалась цена товара и его имя

Слайдер: Назначается на кнопки навигации виджета "Слайдер". Подробнее

Таб: Назначается на кнопки навигации виджета "Таб". Подробнее

Скачать: Указывается, что при клике на элемент началось скачивание файла, только файл должен находиться в папке files

Закрыть модальное: Устанавливается на закрытие модального окна

Закрыть меню: Устанавливается на закрытие меню навигации
