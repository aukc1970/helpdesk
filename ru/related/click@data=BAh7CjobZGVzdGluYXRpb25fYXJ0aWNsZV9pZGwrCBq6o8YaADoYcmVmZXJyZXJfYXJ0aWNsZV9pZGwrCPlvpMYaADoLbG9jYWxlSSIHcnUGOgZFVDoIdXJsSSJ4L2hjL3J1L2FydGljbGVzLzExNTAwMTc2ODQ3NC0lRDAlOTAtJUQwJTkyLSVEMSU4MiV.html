А/В тестирование

Евгений Ворожбянов
  • 21/08/2019 19:45
  • Обновлено

Для создания А/В теста вам необходимо иметь как минимум две страницы, если они у вас есть, тогда зайдите в админ панель CMS Hollpee, выберите ту страницу при переходе на которую, должен включить А/В тест и у этой страницы перейти в "Маркетинг"

Теперь перейдите в вкладку "А/В тест". Сейчас здесь только один вариант страницы, для добавления других вариантов страницы вам нужно нажать на кнопку "Добавить вариант"

Выберите из списка нужные варианты страниц. В данном примере будет добавлено два варианта

Если необходимо выключить один из вариантов, то это делается с помощью кнопки включения и выключения страницы. В данном примере страницу "page_#3" выключена и она не будет показываться в А/В тесте

Для удаления страницы из А/В теста нужно нажать на кнопку "Удалить" напротив этой страницы

Краткая статистика будет отображаться в этом же окне. Вот один из примеров

Принцип работы А/В теста в CMS Hollpee: Когда пользователь заходит на ваш сайт, то у него сохраняется кука, за каждой кукой привязывается в случайном порядке определенная страница из А/В теста, поэтому, когда бы вновь не зашел этот пользователь, то у него всегда будет показываться одна и та же страница из А/В теста. Для того, чтобы увидеть другую страницу, ему необходимо очистить куки в браузере.
